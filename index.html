Deployment Notes

This page is static and ready for Vercel. Replace the Firebase config object with your project values, then publish the `html` folder.

Firebase setup: create a Web App in Firebase, then paste the config values in the firebaseConfig object below. For development, set Firestore security to allow reads/writes while you build the dashboard.

When your Arduino is connected to a gateway (ESP32, NodeMCU, or a local server), use the Firestore document smart-farm/current to keep sensor values and fan state in sync.
